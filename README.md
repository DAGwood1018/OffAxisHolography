# OffAxisHolography
Code for performing off-axis holography. Filtering and phase unwrapping are implemented in C++ and bound to Python using Pybind11. To compile the C++ code, use CMake. C++ dependencies can be installed using the setup bash file. It can be installed as a Python package by navigation to the downloaded repo and executing "pip install .", but this is only automated for a Linux environment currently. 

*This branch is deprecated as it didn't offer much advantage over simply using Python.
